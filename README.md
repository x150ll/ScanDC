# الماسح · Scan

### حوّل أي ورقة إلى مستند نظيف وواضح — مباشرة من جوّالك

## 🚀 كيف تستخدمه؟

<table>
<tr><td>

**1️⃣ &nbsp; اختر مصدرك**
اضغط **التقاط بالكاميرا** لتصوير ورقة، أو **رفع صورة** لاختيار صورة موجودة.

**2️⃣ &nbsp; اضبط الحواف**
سيرسم الماسح إطاراً أخضر تلقائياً حول الورقة. اسحب نقاط الزوايا الأربع لتطابق الحواف بدقة — وستساعدك العدسة المكبّرة.

**3️⃣ &nbsp; اختر الشكل**
جرّب الأنماط الأربعة. **أبيض وأسود** هو الأفضل للمستندات المكتوبة. اضبط السطوع والتباين حسب رغبتك.

**4️⃣ &nbsp; احفظ ونزّل**
اضغط **حفظ** لإضافة الصفحة، ثم افتح قائمة **المستندات** ونزّلها بصيغة **PDF** أو **صور**.

</td></tr>
</table>

> 💡 **نصيحة:** للحصول على أفضل نتيجة، احرص أن تكون الإضاءة جيدة والورقة واضحة داخل الكادر.

<br/>

## 🔤 استخراج النص من الصور

تحويل صورة الورقة إلى نص يمكنك نسخه ولصقه في أي مكان:

1. بعد معالجة المستند، اضغط زر **نص**
2. اختر اللغة: **عربي**، **إنجليزي**، أو **الاثنين معاً**
3. اختر تخطيط الصفحة — جرّب **أعمدة** للأوراق المقسّمة لعمودين
4. سيظهر النص جاهزاً للنسخ خلال لحظات

<tr>
<td width="50%" valign="top">

<td width="50%" valign="top">

## ❓ أسئلة شائعة

<details>
<summary><b>هل تُرفع صوري إلى أي خادم؟</b></summary>
<br/>
لا إطلاقاً. كل المعالجة تتم محلياً داخل جوّالك، ولا يُرسل أي شيء إلى الإنترنت.
</details>

<details>
<summary><b>الكاميرا لا تعمل، ماذا أفعل؟</b></summary>
<br/>
تأكد أنك منحت الموقع إذن استخدام الكاميرا. إن استمرت المشكلة، استخدم خيار <b>رفع صورة</b> بدلاً منها.
</details>

<details>
<summary><b>الإطار الأخضر لا يطابق حواف الورقة؟</b></summary>
<br/>
اسحب نقاط الزوايا الأربع يدوياً لتطابق الحواف، أو اضغط <b>كشف تلقائي</b> ليحاول مرة أخرى.
</details>

<details>
<summary><b>أخطأت في خطوة، كيف أتراجع؟</b></summary>
<br/>
اضغط زر <b>تراجع</b> في شريط الأدوات للرجوع خطوة للخلف.
</details>

<details>
<summary><b>هل يمكنني إيقاف الأصوات؟</b></summary>
<br/>
نعم، من <b>الإعدادات ← الأصوات</b> يمكنك إيقافها متى شئت.
</details>
